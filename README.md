# IA Douane Togo – MVP

## Ce que fait l’outil
- Estimation indicative AVANT importation
- Fourchette de prix (jamais de montant exact)
- Niveau de risque (Faible / Moyen / Élevé)
- Basé sur observations terrain (non officielles)

## Ce que l’outil NE FAIT PAS
- ❌ Aucun calcul douanier officiel
- ❌ Aucune donnée gouvernementale
- ❌ Aucun engagement sur les montants réels
- ❌ Pas de conseil juridique

## Important
Cet outil est une aide à la décision.
Les montants réels peuvent varier selon les contrôles au Port de Lomé.
